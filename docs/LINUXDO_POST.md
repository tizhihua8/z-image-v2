# RyanVan Z-Image - 免费 AI 图像生成服务

## 简介

基于阿里通义万相 Z-Image-Turbo 模型搭建的在线 AI 图像生成服务，使用 Linux DO Connect 登录，根据信任等级分配每日配额。

**体验地址：** https://ryanai.org

## 功能特点

- 🎨 **高质量生图** - 基于 Z-Image-Turbo 模型，4 步采样即可生成高质量图像
- 🔐 **Linux DO 登录** - 使用 Connect OAuth 认证，无需额外注册
- 📊 **按等级配额** - 根据信任等级分配每日生成次数
- 🖼️ **作品广场** - 可将作品分享到公共广场（支持匿名）
- 📱 **响应式设计** - 支持移动端访问

## 配额规则

| 信任等级 | 每日配额 |
|---------|---------|
| 0-1 级 | 1 张 |
| 2 级 | 5 张 |
| 3-4 级 | 20 张 |

每日 UTC 0:00 重置配额。

## 使用方法

1. 访问 https://ryanai.org
2. 点击「Linux DO 登录」授权
3. 同意服务条款
4. 输入提示词，点击生成
5. 等待几秒即可获得图像

## 技术栈

- **前端**：Next.js 14 + TypeScript + Tailwind CSS
- **后端**：FastAPI + SQLAlchemy + SQLite
- **模型**：Tongyi-MAI/Z-Image-Turbo
- **GPU**：本地 RTX 3080 10GB
